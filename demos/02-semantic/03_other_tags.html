details / summary（可折叠内容）

什么是 HTML？

HTML（HyperText Markup Language）是构建网页的标准标记语言。它使用标签来描述网页的结构和内容。

什么是 CSS？

CSS（Cascading Style Sheets）用于控制网页的外观和布局。它可以设置颜色、字体、间距等视觉效果。

默认展开的项目

figure / figcaption（图文组合）

<figure> 表示独立的内容单元（图片、图表、代码片段等），<figcaption> 为其添加说明文字。

time 日期标签

文章发布于：2026年6月4日

活动时间：6月10日下午2:30

营业时间：每天 9:00 - 18:00

datetime 属性提供机器可读的日期格式，便于搜索引擎和浏览器解析。

data 数值标签

当前评分：四星半（共5分满分）

产品编号：豪华套装

<data> 的 value 属性提供机器可读的数值版本。

address 联系方式

联系我们：
邮箱：contact@example.com
电话：+86 400-123-4567
地址：北京市海淀区中关村大街 1 号

<address> 用于提供作者或所属组织的联系信息。

dfn 术语定义

语义化 HTML 指的是使用具有实际含义的 HTML 标签来描述内容，而非仅依赖 <div> 和 <span>。

响应式设计 是一种让网页在不同屏幕尺寸下都能良好显示的设计方法。

<dfn> 标记一个术语的定义位置，可以配合 id 被其他地方引用。

wbr 可选换行机会

下面是一个很长的单词，会在 <wbr> 标记的位置换行：

Supercalifragilisticexpialidocious

拖动浏览器窗口变窄来观察 <wbr> 的换行效果。
