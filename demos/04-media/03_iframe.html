iframe 网页嵌入

基本用法

src — 嵌入页面地址、width/height — 尺寸、srcdoc — 直接内联 HTML

sandbox 安全沙箱

启用 sandbox 后，iframe 默认禁用所有敏感操作。通过 allow-* 逐步开放权限：

取值允许的操作
allow-scripts执行 JavaScript
allow-forms提交表单
allow-same-origin视为同源（可读写 Cookie、localStorage）
allow-popups打开弹窗（如 window.open）
allow-top-navigation修改顶层窗口地址
allow-modals显示模态弹窗
allow-presentation使用 Presentation API

多个值空格分隔：sandbox="allow-scripts allow-forms"

安全注意：不要同时对不受信任的内容使用 allow-scripts 和 allow-same-origin。

loading 懒加载

loading="lazy" 让 iframe 延迟加载，直到滚动到视口附近。默认值为 auto（浏览器自行决定），也可使用 eager（立即加载）。

<iframe src="page.html" loading="lazy" width="400" height="300"></iframe>

name 与 target 配合

点击下方链接，内容会加载到 name="demo-frame" 的 iframe 中：

页面A 页面B

关键属性：name="demo-frame"（给 iframe 命名），<a target="demo-frame">（链接在此 iframe 中打开）。链接使用 data:text/html URI 内联 HTML 内容。

其他属性与兼容

allowfullscreen — 允许 iframe 内容进入全屏模式。

frameborder="0" — 移除默认边框（CSS 中用 border: none 替代更好）。

兜底内容：在 <iframe>...</iframe> 之间写文字，不支持 iframe 的浏览器会显示。
